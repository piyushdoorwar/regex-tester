# RegExp Tester

A fast, offline regex playground with live highlighting, match navigation, and output templating.

## Features
- Live match highlighting over the input text
- Flag toggles (g, i, m, s, u, y)
- Output template support (`$0`, `$1..$n`, `$<name>`, `\n`)
- Match list with search + quick navigation
- Sample loader and clipboard helpers
- Built-in regex cheat sheet

## Run locally
Open `index.html` in your browser. No build step required.

## Project files
- `index.html` UI layout
- `styles.css` theme and layout styles
- `scripts.js` regex logic and interactions
- `favicon.svg` animated SVG favicon
