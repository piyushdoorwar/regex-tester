Regex Cheat Sheet

          
        

        

          

            
Character classes

            
    
              
  • . any char (except newline unless s)
    • 
              
  • \d digit, \D non-digit
    • 
              
  • \w word, \W non-word
    • 
              
  • \s whitespace, \S non-whitespace
    • 
              
  • [abc] any of a, b, c
    • 
              
  • [^abc] not a, b, c
    • 
            

          

          

            
Anchors

            
    
              
  • ^ start of line
    • 
              
  • $ end of line
    • 
              
  • \b word boundary
    • 
              
  • \B not a word boundary
    • 
            

          

          

            
Quantifiers

            
    
              
  • * 0 or more
    • 
              
  • + 1 or more
    • 
              
  • ? 0 or 1
    • 
              
  • {n} exactly n
    • 
              
  • {n,} at least n
    • 
              
  • {n,m} between n and m
    • 
            

          

          

            
Groups and alternation

            
    
              
  • (...) capturing group
    • 
              
  • (?:...) non-capturing group
    • 
              
  • (?<name>...) named group
    • 
              
  • | alternation
    • 
            

          

          

            
Lookarounds

            
    
              
  • (?=...) positive lookahead
    • 
              
  • (?!...) negative lookahead
    • 
              
  • (?<=...) positive lookbehind
    • 
              
  • (?<!...) negative lookbehind
    •

    • 
              
  • m multiline
    • 
              
  • s dot matches newline
